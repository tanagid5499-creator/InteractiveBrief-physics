เปิดจอครู

แสดงคำถาม + แท่งนับสด + ปุ่มคุม (ถัดไป/เปิด-ปิด/ล้าง) — เปิดบนจอโปรเจกเตอร์

เปิดจอนิสิต

กดเลือก A/B/C ส่งคำตอบ — เปิดอีกแท็บ/หน้าต่าง (จำลองมือถือนิสิต)
วิธีลอง (เดโม):
  1. เปิด จอครู หนึ่งแท็บ และ จอนิสิต อีกแท็บ (เบราว์เซอร์เดียวกัน)
  2. กดตอบในจอนิสิต → เห็นแท่งนับขยับในจอครูทันที
  3. ครูกด "ถัดไป" เพื่อเปลี่ยนข้อ · "ปิดรับคำตอบ" เพื่อหยุด · "ล้างเซสชันใหม่" สำหรับคลาสถัดไป
ข้อจำกัดของเดโม: ใช้ BroadcastChannel = สื่อสารได้เฉพาะ เบราว์เซอร์เดียวกันในเครื่องเดียว (ลองกลไก/ดีไซน์) · เวอร์ชันใช้จริงในคลาส (มือถือนิสิตหลายเครื่อง) จะต่อ Supabase + QR + deploy — โครงข้อความเหมือนเดิม สลับแค่ตัวส่ง-รับ
